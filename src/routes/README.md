# Routes

This directory defines the API route definitions. It maps HTTP endpoints to specific controller methods and applies necessary middleware.

## Files

-   `pdf.routes.ts`: API routes for PDF operations (e.g., `POST /api/pdf/convert`).
-   `auth.routes.ts`: Authentication routes (login, register, verify).
-   `dashboard.routes.ts`: Protected routes for the user dashboard.
-   `landing.routes.ts`: Public routes for landing pages and tool showcases.
-   `user.routes.ts`: Routes for user profile management.
-   `billing.routes.ts`: Routes for subscription and billing.
-   `services.routes.ts`: Routes for managing available services/tools.
