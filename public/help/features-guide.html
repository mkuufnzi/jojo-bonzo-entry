Features Guide

Controlling access to tools.

1. What is a Feature?

A Feature is a "Key" that unlocks a specific part of the app.

Example: "AI Generation"

If a user has this key (via their plan), they can use the AI tools.

2. Feature Types

Boolean (On/Off): You either have it or you don't. (e.g., "Remove Watermark").

Metered (Counted): You have it, but with a limit. (e.g., "50 Documents / Month").

3. Connecting Services

To make a Service billable, go to the Services page and link it to a required Feature Key.

Example: Link "PDF Tool" service to "pdf_generation" feature. Only users with that feature can access it.
