Plans Guide

Managing subscriptions and pricing.

1. Plan Types

Active: Publicly visible plans that users can buy right now.

Archived: Old plans. Existing users keep them, but new users can't see them.

Pricing is synced from Stripe. To change a price, change it in Stripe first, then click "Sync" here.

2. Features & Limits

A plan is just a bundle of features.

  • Features: Click "Edit" on a plan to choose what tools it unlocks.
  • Quotas: Set usage limits (e.g., "50 PDF generations/month") per tool.
Safety First

Changing a plan's features affects ALL current subscribers immediately. Be careful removing features!
