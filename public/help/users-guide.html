Users Guide

1 Overview

This dashboard manages all user accounts and access control.

Three Tabs:

  • End Users - Customer accounts
  • System Users - Admin/staff accounts
  • Banned IPs - Blocked addresses

2 System Roles

ROOT Full Access

Complete system control. Can manage all users, services, billing, and system settings. Highest privilege level.

CEO Executive View

View revenue, billing, subscriptions, analytics, users, and features. Read-only executive dashboard access.

COO Operations

View revenue, subscriptions, analytics, users, and features. Operations oversight without edit access.

DEVOPS Technical

System management, logs, analytics, service configuration. Can ban users/IPs and manage service settings.

MARKETING Growth

View analytics, manage features and plans. Focused on product growth and feature releases.

SUPPORT Customer Care

View users only. Limited access for customer support and account assistance.

USER Customer

Standard customer account. No admin panel access. Uses public-facing features only.
